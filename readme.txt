## Nombre del programa
MiCalculadora

## Descripción
MiCalculadora es una aplicación que permite realizar operaciones matemáticas básicas, incluyendo suma, resta, multiplicación, división, potencias y raíces cuadradas. La interfaz está construida con CustomTkinter, ofreciendo una experiencia de usuario moderna y visualmente atractiva. Además, cuenta con una función de deshacer/rehacer y soporte para la evaluación de expresiones matemáticas complejas.

## Requerimientos
- Python 3.x
# Librerías:
- customtkinter: Para la interfaz gráfica de usuario.
- tkinter: Para el manejo básico de la interfaz.
- Libraries:
Puedes instalar las dependencias necesarias utilizando el siguiente comando:

pip install customtkinter

## Instalación
Clona el repositorio o descarga los archivos del programa.
Asegúrate de tener las librerías mencionadas en "Requerimientos" instaladas.
Coloca el archivo del ícono (IconoMiCalculadora.ico) en una carpeta llamada Iconos dentro del mismo directorio del programa.

## Instrucciones de uso
- Ejecuta el archivo main.py para iniciar la calculadora.
- Utiliza los botones en pantalla para ingresar números y operadores.
- Los resultados de las operaciones aparecerán en la barra de salida, sin eliminar la expresión mostrada en el cuadro de entrada.
- Utiliza los botones ⬅️ y ➡️ para deshacer o rehacer las operaciones.
- El botón C limpia la entrada y el resultado, y el botón ⌫ elimina el último carácter ingresado.
